# Amiibo API

# Описание
Приложение для просмотра существующих фигурок/карточек/ярнов Amiibo по играм, выходившим на консоли Nintendo

# Архитектура 
* Архитектура приложения - #Clean Architecture
* Архитектура UI - #MVVM

# Используемые технологии
* OkHttp 4
* Moshi  
* RxJava 2
* Architecture Components
* Picasso
